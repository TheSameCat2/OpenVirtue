# OV-20260701-scheduler-if-start-cycle: IF_START Versus First EACH_TICK Dispatch
Date: 2026-07-01
Status: proposed
Observer: OpenVirtue maintainers
Implementer: OpenVirtue maintainers

## Question
When `IF_START` assigns an action into `EACH_TICK.1`, does that scheduled action
run after startup, and how soon is the first dispatch observable?

## Why It Matters
OpenVirtue currently has a provisional scheduler API. Exact startup-versus-tick
ordering determines how global scripts observe initialized skills and affects
movement, animation, AI hooks, and any per-frame script logic.
