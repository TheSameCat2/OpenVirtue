# OV-20260701-scheduler-if-start-cycle: IF_START Versus First each_cycle
Date: 2026-07-01
Status: proposed
Observer: OpenVirtue maintainers
Implementer: OpenVirtue maintainers

## Question
Does the original runtime execute the level startup action (`IF_START`) before
the first scheduled `each_cycle` action?

## Why It Matters
OpenVirtue currently has a provisional scheduler API. Exact startup-versus-cycle
ordering determines how objects observe initialized global skills and affects
movement, animation, AI hooks, and any per-frame script logic.
